## 项目简介

本项目是基于 **Spring Cloud + Vue3** 微服务架构的医院预约挂号管理系统，包含以下核心功能：
- 科室管理
- 医生管理
- 患者管理
- 排班管理
- 预约挂号
- 病历管理
- 系统公告

### 技术栈

**后端**
- Spring Boot 2.7.18
- Spring Cloud 2021.0.9
- Spring Cloud Alibaba 2021.0.6.1
- Nacos（注册/配置中心）
- Sentinel（流量控制）
- MySQL 5.7+
- Redis
- Java 17
- Maven 3.6+

**前端**
- Vue 3.5.16
- Element Plus 2.10.7
- Vite 6.3.5
- Pinia 3.0.2
- Node.js 16+

## 环境准备

### 必须安装的软件

1. **JDK 17**
   - 下载地址：https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
   - 配置JAVA_HOME环境变量

2. **Maven 3.6+**
   - 下载地址：https://maven.apache.org/download.cgi
   - 配置MAVEN_HOME环境变量
   - 配置settings.xml（建议使用阿里云镜像）

3. **Node.js 16+**
   - 下载地址：https://nodejs.org/
   - 安装后验证：`node -v` 和 `npm -v`

4. **MySQL 5.7+**
   - 下载地址：https://dev.mysql.com/downloads/mysql/
   - 默认端口：3306

5. **Redis**
   - 下载地址：https://redis.io/download
   - Windows版推荐：https://github.com/microsoftarchive/redis/releases
   - 默认端口：6379

6. **Nacos 2.x**
   - 下载地址：https://github.com/alibaba/nacos/releases
   - 默认端口：8848

### 可选软件

- **Docker & Docker Compose**（用于容器化部署）
  - 下载地址：https://www.docker.com/get-started

## Docker方式部署（推荐）

### 1. 进入Docker目录

```bash
cd e:\GraduationDesign\后端源代码\RuoYi-Cloud\docker
```

### 2. 启动基础设施服务

```bash
docker-compose up -d ruoyi-mysql ruoyi-redis ruoyi-nacos
```

### 3. 导入数据库

- 创建数据库：`ry-cloud`
- 导入SQL文件：`e:\GraduationDesign\后端源代码\RuoYi-Cloud\sql\hospital_booking_new.sql`
- 导入初始化数据：`e:\GraduationDesign\后端源代码\RuoYi-Cloud\init_admin_data.sql`

### 4. 配置Nacos

访问Nacos控制台：http://localhost:8848/nacos
- 默认账号：nacos
- 默认密码：nacos

在配置中心导入配置文件：
- 将 `e:\GraduationDesign\nacos配置\nacos\` 目录下的所有yml配置文件导入到Nacos

### 5. 构建并启动后端服务

```bash
# 先编译后端项目
cd e:\GraduationDesign\后端源代码\RuoYi-Cloud
mvn clean package -DskipTests

# 复制jar包到docker目录（参考docker目录下的说明）
# 然后启动所有服务
cd docker
docker-compose up -d
```

### 6. 启动前端

```bash
cd e:\GraduationDesign\前端源代码\RuoYi-Cloud-Vue3
npm install
npm run dev
```

## 常见问题

### 1. Maven编译失败

**问题**：依赖下载失败
**解决**：
- 检查Maven配置是否正确
- 配置阿里云镜像：

```xml
<mirror>
  <id>aliyunmaven</id>
  <mirrorOf>*</mirrorOf>
  <name>阿里云公共仓库</name>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

### 2. Nacos配置未生效

**问题**：服务启动后读取不到配置
**解决**：
- 确认Nacos已正常启动
- 检查配置文件的Data ID和Group是否正确
- 确认配置文件格式正确（yml格式）

### 3. 数据库连接失败

**问题**：Communications link failure
**解决**：
- 确认MySQL服务已启动
- 检查Nacos配置中的数据库连接信息
- 确认数据库用户权限

### 4. 前端无法访问后端接口

**问题**：跨域或接口404
**解决**：
- 确认Gateway服务已启动
- 检查前端配置的API地址是否正确
- 查看浏览器控制台Network标签的错误信息

### 5. 端口被占用

**问题**：Address already in use
**解决**：
- Windows下查找占用进程：`netstat -ano | findstr "端口号"`
- 结束进程：`taskkill /PID 进程ID /F`
- 或修改配置文件中的端口号

### 6. 内存不足

**问题**：Java heap space
**解决**：
- 修改启动脚本中的JVM参数，减小Xmx值
- 关闭不必要的服务
